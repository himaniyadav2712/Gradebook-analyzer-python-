Gradebook Analyzer
Author: Himani Yadav
Date: November 4, 2025

Description
The Gradebook Analyzer is a simple command-line Python program that allows users to manually enter student names and their marks. It automatically calculates key statistics, assigns letter grades, and displays a summary of the results.
This program is designed for teachers, tutors, or students who want a quick way to analyze performance without relying on spreadsheets.

Features
Manual data entry for student names and marks.
Error handling for invalid inputs and marks outside the 0–100 range.
Statistical analysis including:
Average score
Median score
Highest and lowest marks
Automatic grade assignment (A–F) based on score thresholds.
Grade distribution summary showing how many students fall into each grade.
Pass/fail summary (pass mark = 40).
Clean and formatted results table.
